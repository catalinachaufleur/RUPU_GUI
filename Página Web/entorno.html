Entorno de trabajo

En esta sección se revisarán todos los programas y librerías requeridas para manejar la plataforma RÜPÜ.

Instalar programas

Comienza descargando e instalando el IDE de Arduino. Es importante que tengas en cuenta que RÜPÜ soporta hasta la versión 2.1.1, nuevas versiones deberán ser verificadas. De igual forma, descarga e instala Python, en este caso, RÜPÜ tiene soporte hasta la versión 3.x.x.

Set up Arduino

En esta sección se realizará la configuración del IDE de Arduino para poder cargar los códigos a las placas ESP32. Esta configuración contempla el gestor de tarjetas y las librerías que se utilizarán en el código.

Gestor de tarjeta

El gestor de tarjeta es la herramienta que permite utilizar los módulos ESP32 a través del IDE de Arduino, utilizaremos la versión 1.0.4 del gestor esp32 creado por Espressif Systems.

  1. Dirígete a File>Preferences
  2. Pega el siguiente link en "Additional Boards"
    3. https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
  3. Ve al gestor de tarjetas y busca esp32 Espressif
  4. Instala la versión 1.0.4

Librerías

El código a utilizar para controlar los robots requiere de la instalación de las siguientes librerías:

Librería Autor Versión
VL53L0X Pololu 1.3.1
analogWrite ErroPix 0.1
Encoder Paul Stoffregen 1.4.2

Set up Python

Para utilizar la interfaz gráfica de usuario provista para RÜPÜ es necesario instalar ciertas librerías antes de correr el código. Por esta razón se utilizará pip install para realizar la descarga e instalación:

Las librerías a instalar corresponden a CustomTKinter , Matplotlib y Paho-mqtt. Las cuales pueden ser instaladas a través del instalador de paquetes de python pip. 


    $ pip install customtkinter
    $ pip install matplotlib
    $ pip install paho-mqtt
